🔧 CONFIGURACIÓN OPCIONAL
═══════════════════════════════════════════════════════════

Para habilitar Google OAuth:

1. Ve a: https://console.cloud.google.com/
2. Crea proyecto y credenciales OAuth
3. Copia el Client ID
4. Edita: Frontend/src/environments/environment.ts
5. Pega tu Client ID en: googleClientId


❓ SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════

Problema: "ng no se reconoce como comando"
Solución: npm install -g @angular/cli

Problema: "Puerto 4200 en uso"
Solución: ng serve --port 4201

Problema: "Puerto 3000 en uso"
Solución: Edita server-simple.js, cambia PORT = 3000

Problema: "Error al hacer login"
Solución: Verifica que el backend esté corriendo

Problema: "Google OAuth no funciona"
Solución: Configura el Client ID (ver arriba)
